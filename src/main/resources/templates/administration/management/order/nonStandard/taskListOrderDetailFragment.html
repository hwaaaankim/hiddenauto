주문 상세 수정 ORDER

목록 화면에서 오더의 금액, 배송정보, 현장주소, 담당자, 옵션, 첨부파일을 바로 수정합니다.

Task 상세 Order 상세
수정 안내
현장배송 선택 시 현장주소가 표시되며 필수로 입력해야 합니다. 직배송, 화물, 현장배송은 배송팀 담당자 지정이 필수입니다. 그 외 배송수단은 담당자 지정이 선택사항이며, 담당자가 없으면 배송순서에서 제외됩니다.
대리점을 변경하면 신청자, 주문자, 등록 배송지 선택 기준도 함께 변경됩니다.
신청자를 비워두고 대리점만 선택하면 대표회원 기준으로 저장됩니다.
취소 상태로 저장하면 배송순서 정보는 자동 삭제됩니다.
생산팀이 확인해야 하는 분류입니다.
금액 계산

총비용과 부가세는 공급가 기준으로 계산됩니다. 공급가가 0이면 부가세와 총비용도 0입니다.

자동계산
담당자가 지정된 경우 이 날짜 기준으로 배송순서가 관리됩니다.
현장배송 선택 시 현장주소가 표시됩니다. 직배송, 화물, 현장배송은 담당자 지정이 필수입니다.
직배송, 화물, 현장배송은 필수입니다. 그 외 배송수단은 선택사항입니다.
현장배송이 아닌 경우 필수 주소입니다. 대리점 기본주소, 등록 배송지, Daum 주소검색으로 입력할 수 있습니다.
주소
배송수단이 현장배송인 경우에만 사용됩니다. 현장배송에서 다른 배송수단으로 변경하면 저장 시 현장주소는 삭제됩니다.
